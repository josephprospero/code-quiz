# Joseph Prospero's JavaScript Quiz
Here is my quiz based off of JavaScript. It features HTML, JavaScript, and Bootstrap. You are given 3 minutes to complete the quiz and if you answer the question incorrectly, you are penalized 5 seconds. There is also a list of high scores that allows you to save your score along with your initials.
